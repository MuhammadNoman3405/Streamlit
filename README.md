# üçé Streamlit Fruit & Age App
This is a practice project built using **Streamlit** and **Python**.

## Features
* **Fruit Inventory Dashboard:** Analyzes fruit cost, revenue, and expiry dates.
* **Age Calculator:** Calculates current age and days remaining until the next birthday.
* **Interactive UI:** Includes voting polls, sidebars, and expanders.
**Live Currency Converter:** convert the currency by entering the amount in the pkr.

## How to Run
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`.
3. Run the app: `streamlit run main.py`.
